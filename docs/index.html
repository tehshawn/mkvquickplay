MKV QuickPlay

QuickLook-style instant video preview for macOS, powered by mpv.

Download for macOS View on GitHub
brew install --cask tehshawn/tap/mkvquickplay

Why?

macOS QuickLook can't preview MKV and many other formats — and Quick Look plugins only show you one file at a time. MKV QuickPlay is built for the folder: select a video in Finder, press Cmd+Shift+V, and it opens immediately in an mpv window. Arrow through the folder in Finder's sort order, send junk to the Trash, and undo — all without leaving the keyboard. Nothing else on macOS does keyboard-driven video culling.

Features

⚡ Instant preview

Cmd+Shift+V on any selected video in Finder.

🎞️ Native resolution

Opens at the video's real dimensions; only oversized videos shrink to fit.

⬆️⬇️ Keyboard navigation

Step through the folder in Finder's sort order; the selection follows along.

🗑️ Cull & undo

Delete sends a clip to the Trash and advances; Cmd+Z brings it back.

🧩 Wide format support

MKV, AVI, WebM, MP4, M4V, MOV, WMV, FLV, TS, MTS, M2TS.

🪶 Lives in the menu bar

No Dock icon, optional launch at login, recordable shortcut, auto-updates, signed & notarized.

Requirements

macOS 13 (Ventura) or later, and mpv (brew install mpv — installed automatically with the Homebrew cask).
